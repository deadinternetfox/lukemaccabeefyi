üèóÔ∏è System Architecture

Bot Layer: Python aiogram bot handles all user interactions, subscription management, and config delivery.

Backend API: FastAPI service manages user data, payment verification, and XRay configuration.

Proxy Layer: XRay instances running in Docker containers, managed via X-UI panels.

Payment Layer: Cryptocurrency payment processor with webhook callbacks for automatic activation.

üßó Challenges & Solutions

Challenge: Protocol Blocking

Some networks actively block VPN protocols using deep packet inspection.

Solution: Implemented multiple protocols and obfuscation techniques. Users can switch protocols if one is blocked.

Challenge: Config Distribution

QR codes and config files need to work across many different client apps.

Solution: Built a config generator that creates compatible URIs for all major VPN clients on iOS, Android, Windows, and macOS.

Challenge: Payment Verification

Crypto payments need reliable verification before activating subscriptions.

Solution: Integrated with payment APIs that provide webhook callbacks, with fallback manual verification for edge cases.

üîê Security Considerations

Zero Logging Policy

No connection logs are stored. Only minimal subscription data is kept for service operation.

Encrypted Configs

All configuration files use strong encryption and unique per-user credentials.

Regular Key Rotation

Server keys are rotated regularly to maintain security even if a config is compromised.
